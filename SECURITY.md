# Security Policy

## Reporting a Vulnerability

If you believe you've found a security vulnerability in any of the packages in this repository, please report it responsibly.

**Email**: security@here.build

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested mitigation, if any

We aim to acknowledge reports within 72 hours and to provide a coordinated disclosure timeline based on severity.

## Supported Versions

This repository is in active development prior to its 1.0 release. Security fixes are made against the current `main` branch; users are encouraged to stay current.
